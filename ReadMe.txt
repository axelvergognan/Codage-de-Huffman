Voici les étapes pour faire fonctionner l'agorithme de Huffman :

1 - Pour coder un texte il faut le copier/coller dans le fichier "texte-à-coder.txt"
    puis sélectionner "1" pour crypter le texte lisible.

2 - Pour décoder un texte crypté illisible, même manipulation,
    il faut le copier/coller dans le fichier "text-à-decoder.txt" puis sélectionner "2" pour décrypter le code en texte lisible.

Attention: votre texte doit se faire sur une seule ligne donc éviter les retours à la ligne.
